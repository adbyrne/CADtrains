# CLAUDE.md — CokeOvens

Project-specific guidance for the CokeOvens parametric CAD model.

## Overview

HO scale (1:87) modular bank coke oven bank for a model railroad layout. C&O Railway
prototype, circa 1900 West Virginia coalfields. All geometry is generated by
`scripts/generate_cokeovens.py` via the FreeCAD MCP bridge at `localhost:9875`.

**Current version:** v8 (production, see `docs/DESIGN.md` for full spec)

## Resuming Work

1. Load `freecad/CokeOvens.FCStd` in FreeCAD (or let the script recreate it)
2. Run `scripts/generate_cokeovens.py` via the MCP bridge:
   ```python
   proxy.execute(open("scripts/generate_cokeovens.py").read())
   ```
3. STLs are exported to `printed_files/` automatically by the script

## Script Architecture

`generate_cokeovens.py` builds four Part objects in a single FreeCAD document:

- **`CokeOvenSection`** — 4-oven section (primary print piece)
- **`CokeOvenSection_1`** — 1-oven test piece (same code, N_OVENS=1)
- **`EndCapStoneWall`** — left bank end cap
- **`EndCapStoneWall_Mirror`** — right bank end cap (X-mirror of above)

Key construction order in `make_section()`:
1. Base slab + alignment peg (right face) + alignment hole (left face)
2. Parapet lip
3. Front wall
4. Dome shells (hollow hemispheres, raised by ARCH_RAISE)
5. Back shelf with N-scale larry track grooves
6. Inter-dome fill piers
7. Slope fill (trapezoidal wedge, domes cut from fill)
8. **FUSE** all parts → `section`
9. Arch cuts (rectangular reveal + semicircular opening)
9.5. Retaining wall stone courses (post-fuse — must stay here to avoid parapet fill-back)
10. Arch voussoir joints (radial + extrados groove + jamb joints)
11. Front wall stone courses (post-fuse, arch rings excluded via Z-aware chord masking)
12. Print-assist fillet (1.5 mm at slope fill / front wall junction)

**Critical ordering rule:** Stone courses on the retaining wall and front wall MUST be
applied post-fuse (steps 9.5 and 11). Applying pre-fuse allows parapet/base-slab
material to fill back the grooves during boolean fuse operations.

## Parameters to Tune After Print Test

| Parameter | Current | Adjust if |
|-----------|---------|-----------|
| `STONE_MORTAR` | 0.42 mm | Courses not visible → increase to 0.5 mm |
| `STONE_GD` | 0.9 mm | Grooves lost in print → increase to 1.0–1.1 mm |
| `PEG_HOLE_R` | 1.15 mm | Peg too tight → increase; too loose → decrease |
| `LARRY_GW` / `LARRY_GD` | 1.8 / 0.9 mm | Track ties don't seat flush → adjust |

## FreeCAD Notes

- All geometry uses Part module (not PartDesign) — boolean operations throughout
- `stone_courses_y()` and `stone_courses_x()` generate mortar groove cut tools
- `MASTER_COURSES` (seed=42) provides Z-aligned course heights used by all faces
- `CAP_YARD_H` is computed (not fixed) as the next course boundary above `YARD_BASE_H`
